## 62. 不同路径

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为“Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为“Finish”）。

问总共有多少条不同的路径？
### 解题
递归
```javascript
var uniquePathsV1 = function(m, n) {
    if (m === 1 || n ===1) return 1
    return uniquePaths(m-1, n) + uniquePaths(m, n-1)
};
```

动态规划
```javascript
var uniquePaths = function(m, n) {
    let dp = new Array(n)
    for(let i = 0; i < n; i++) {
        dp[i] = new Array(m)
    }
   for(let j = 0; j < m; j++) {
       dp[0][j] = 1
   }
   for(let g = 0; g < n; g++) {
       dp[g][0] = 1
   }
   for (let i = 1; i < n; i++) {
       for (let j = 1; j <m; j++) {
           dp[i][j] = dp[i-1][j] + dp[i][j-1]
       }
   }
   return dp[n-1][m-1]
};
```
